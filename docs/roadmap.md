# Roadmap

Possible v2 improvements:

- Persistent log offsets across restarts
- Optional JSON structured logs
- Prometheus metrics endpoint
- Per-monitor enable/disable config
- Concurrent website checks with a bounded worker pool
- Journald backend for SSH detection on systems without `/var/log/auth.log`
- Better Nginx parser support for custom log formats
- Alert allowlists and denylists
- Debian package
- Arch package
- Dedicated non-root service user guide
- More complete unit tests
